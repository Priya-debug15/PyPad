# Description
A **basic text editor application** built using Python and Tkinter. It allows users to **create, edit, save, and open text files** in a simple graphical interface.

# Features
- Text area for writing and editing
- Menu options: New, Open, Save, Exit
- Basic editing functions: Cut, Copy, Paste
- Scrollbars for navigation

# Learning Outcomes
- GUI programming with Tkinter
- File handling in Python
- Event-driven programming
